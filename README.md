# Spine-android-demo

可以直接使用的Spine动画android库，其中的gdx库已经升级为1.9.14版本，并且支持Texture渲染.

支持的Spine动画资源 json版本中会有如下字段：
```
"skins": [
    {
      "name": "default",
      "attachments": {
        "zuoyan": {
          "images/zuoyan": {
            "x": 33.68,
            "y": -19.63,
            "rotation": -92.17,
            "width": 33,
            "height": 33
          }
        }
      }
    }
  ]
  ```
